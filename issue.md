## 各種情報
サービスの概要とMVPで検証する事柄：
日本の行事・行事食・季節風呂などをカレンダー形式で表示し、それらの情報を通して「季節の移ろい」を感じたり、実践できたりするかを検証する。
サービス名：めぐる暦～日々をつむぐ季節と行事
サービスURL：https://meguru-koyomi-mi4y.onrender.com/

## READMEレビュー時にMVPリリース時点で実装予定だった機能
- [x] カレンダーの表示
- [x] 季節湯の表示（サイドバー＆カレンダー）
- [x] 行事の表示（サイドバー＆カレンダー）
- [x] モーダルの表示（季節湯＆行事）

## READMEレビュー時から変更した機能
季節湯と行事のアイコン（MVPでは表示なし）
季節湯と行事のイメージ画像（MVPではデフォルト画像もしくは画像なしにとどめる）
